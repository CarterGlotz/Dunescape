# Soul

## Core promise

Solara: Sunfall is a browser RPG where every player's death matters to the whole community. The sun is dying. Only collective play can save it — or let it burn out.

## Non-negotiables

- must always be true: The shared sun mechanic is the heart of the game — every death must feel meaningful beyond the individual player
- audience should feel: Invested in a world that responds to their presence; curious about the Oracle; driven by the tension between Sunkeepers and Eclipsers
- quality bar: VaultSpark Studios standard — no half-shipped features; every mechanic must be playable end-to-end

## Anti-goals

- must never drift into: Generic idle game with no emotional stakes
- cheap imitation to avoid: OSRS clone without its own identity (Phase 0 IP cleanup completed)
- tonal failure mode: Cold, mechanical game loop with no world soul

## Creative pillars

1. **Collective consequence** — individual actions shape a shared world state
2. **Permanence** — graves, relics, and Oracle broadcasts outlive any single session
3. **Mystery** — the Oracle knows more than it reveals; the sun's history is hidden in lore fragments
4. **Faction identity** — Sunkeepers vs Eclipsers creates ideological stakes and social sharing
