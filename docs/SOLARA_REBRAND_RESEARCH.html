1. Context — What We Are Building From

Current Game: "Dunescape" (being retired)

  • Stack: React + Vite, single-page browser game, canvas rendering at 544×448px (17×14 tiles × 32px)
  • Codebase: ~2,234 lines in src/App.jsx — all game logic in one file, all constants inline
  • World: 100×100 tile map, desert theme, pixel art aesthetic
  • Systems built and functional: 21 skills, quest system (13 quests), combat (melee/ranged/magic), prayer system, inventory/bank/equipment, NPC dialogue, monster system with boss variants, crafting/smithing/herblore/fletching, firemaking, cooking, farming, runecrafting, dungeon instances, arena wave mode, prestige system, pets, daily challenges, faction reputation, player camp, lore codex, side quests, QP unlock tree, character appearance, combo meter, async ghost dungeon (partial)
  • Save system: localStorage key dunescape_save, SAVE_VERSION=4
  • Deployment: GitHub Pages via VaultSparkStudios/VaultSparkStudios.github.io at /dunescape/

Why Rebrand?

  • Trademark risk: "Dunescape" combines "-scape" (Jagex/RuneScape trademark signature) with "Dune" (Frank Herbert estate active IP). Double conflict exposure.
  • Differentiation: The game has evolved well beyond a RuneScape clone — it deserves its own identity
  • Growth ceiling: The "Dunescape" name signals derivative product to press, platforms, and players

New Name: SOLARA

IP Status: CONFIRMED CLEAR
USPTO Class 9 (software) and Class 41 (entertainment services) searches show no active gaming or entertainment conflicts. "Solara" marks exist only in pharmaceutical (Cytel Inc.) and financial services — no gaming overlap. Recommended: file USPTO intent-to-use application before launch. Attorney clearance search cost: $500–2,000. Can file up to 3 years before release.
  • Avoid in naming: "Solar Empire" phrasing (Sins of a Solar Empire trademark), "Solarpunk" as primary name (early-access game dominates that search term)
  • Safe territory: "Solara: [Subtitle]" as a combined mark is clean and highly trademarkable

3. Research Findings Summary

Browser Game Market (2025–2026)

  • Browser games market projected to reach $9.07B by 2030 (3.1% CAGR)
  • HTML5/WebGL enabling near-AA quality in-browser — resurgence underway
  • Trending NOW: low-fidelity social chaos games ("friendslop"), cozy games, daily puzzle format, idle-roguelite hybrids
  • 42M+ sessions achieved by one browser game in Q3–Q4 2025 with minimal marketing spend — organic sharing drove it
  • 70% of acquisition from social shares vs. paid ads in top-performing browser games

What Makes Games Viral

  • Shareable artifact: Something players send — emoji score card, seed code, screenshot, challenge URL
  • Daily hook: Same puzzle/dungeon for all players globally = Wordle effect. Players share results to compare
  • Identity investment: "My grave lives in that world" / "I named that canyon" / "My clan won this week" = emotional ownership
  • Messaging app > public social: Players prefer sharing to Discord/WhatsApp over Twitter — messaging app shares generate the most installs
  • Variable reward: Unpredictable drops, run outcomes, seed results = slot machine psychology (ethical version)
  • Wordle's viral math: player counts doubled weekly at peak — each shared emoji card = free ad to hundreds of followers

Claude-Only Build Ceiling

  • Confirmed realistic ceiling: 10,000–20,000 lines of game code (current codebase: ~2,234 lines — well within range)
  • Stretching but feasible: 30,000–40,000 lines with good architecture documentation
  • Current single-file architecture needs to be split into modules as it grows past ~5,000 lines

  • Sweet spots: idle/clicker, puzzle, turn-based, roguelite with preset content, browser canvas games
